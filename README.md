# Features
 - Point Charges
 - Line Segment Charges
 - Infinite Plane Charges
 - Ring Charges
 - Dish Charges
 - Washer Charges
 - interactive transformation gizmos for editing the scene
 - Real-time updating of the Electric Field using WebGL
